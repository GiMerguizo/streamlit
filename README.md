# Projeto com o Streamlit

- **Documentação:**[https://streamlit.io/](https://streamlit.io/)

## Requisitos

1. Instalação do Streamlit
```bash
pip install streamlit
```

2. Instalação das bibliotecas do projeto
```bash
pip install yfinance
```

## Executar o projeto
No CMD:
```
streamlit run main.py
```
